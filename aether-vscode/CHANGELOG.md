# Changelog

## 0.2.0 — 2024-04-03

### Added
- LSP server integration (IntelliSense, hover, go-to-definition, live diagnostics)
- Debug Adapter Protocol (DAP) support — breakpoints, stepping, variable inspection
- Run button (Ctrl+Shift+R) and Debug button (F5) in editor title bar
- REPL command — opens Aether REPL in integrated terminal
- Type check command — runs `aether check` on current file
- Status bar with Aether run button for .ae files

## 0.1.0 — 2024-04-03

### Added
- Initial release
- Syntax highlighting for all Aether keywords, operators, types
- String interpolation highlighting
- 10 code snippets
- Bracket matching and auto-closing
- Comment toggling
