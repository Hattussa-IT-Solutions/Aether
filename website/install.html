Installation

From Source (All Platforms)

git clone https://github.com/aether-lang/aether.git
cd aether
cargo build --release
./target/release/aether --version

Requirements

  • Rust 1.70+ (install via rustup.rs)
  • macOS, Linux, or Windows

VS Code Extension

cd aether-vscode
npm install
npx vsce package
# Install the .vsix file in VS Code

Verify Installation

aether --version
aether run examples/hello.ae
