Aether

A modern programming language with built-in parallelism, pattern matching, genetic evolution, and GPU compute. Expressive. Fast. Beautiful.

Get Started Try Online 
// Hello World in Aether
print("Hello from Aether!")

// Pattern matching
def classify(score) {
    match score {
        90..100 -> "A"
        80..90  -> "B"
        _       -> "F"
    }
}

// Parallel execution
parallel {
    users  = fetch_users()
    orders = fetch_orders()
}

// Functional pipelines
result = data |> transform |> filter |> collect

Expressive Syntax

Clean, Python-inspired syntax with braces. No semicolons, no parentheses around conditions. String interpolation built in.

Built-in Parallelism

parallel {} blocks run tasks concurrently. Race conditions, timeouts, and dependencies are first-class constructs.

Pattern Matching

Powerful match expressions with destructuring, range patterns, guards, and exhaustiveness checking.

Modern OOP

Classes, structs, enums, interfaces, plus novel concepts: reactive properties, temporal state, weave injection.

GPU Compute

@gpu decorator compiles functions to CUDA PTX, WebGPU WGSL, or ROCm HIP. Tensor operations with autodiff.

Genetic Evolution

genetic classes with chromosomes, genes, and fitness functions. Built-in evolve blocks for optimization.
